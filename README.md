🖋️ Слово Незалежних | Літературний Квест

Інтерактивний веб-додаток у форматі вікторини, створений спеціально до Всесвітнього дня письменника. Проєкт спрямований на популяризацію української літератури серед студентів та молоді через сучасний гейміфікований інтерфейс.

✨ Особливості проекту

12 інтерактивних завдань: цитати від Григорія Сковороди до Сергія Жадана.

Сучасний дизайн: чистий інтерфейс у національних кольорах з використанням скляних ефектів (Glassmorphism).

Система підказок: кожне питання має унікальну підказку з цікавим фактом про автора.

Адаптивність: додаток коректно відображається на смартфонах, планшетах та десктопах.

Технічна чистота: проєкт реалізовано в одному файлі index.html без зайвих важких фреймворків.

🛠 Технологічний стек

HTML5 / CSS3

Tailwind CSS (через CDN)

Vanilla JavaScript (логіка гри, алгоритм перемішування Фішера-Єйтса)

Google Fonts (Montserrat та Playfair Display)
